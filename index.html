Piece+

A beginner-friendly programming language that bridges Python's simplicity with C's structure. Learn to code with confidence.

View on GitHub See Examples

What it offers

Language Features

Piece+ is designed for progressive learners — start simple and grow into structured, statically-typed code.

🔤

Optional Static Typing

Declare variables with int, float, or str — or use let for dynamic typing.

🔒

Compile-Time Safety

Type errors, undeclared variables, and duplicate declarations are caught before your code ever runs.

⚡

Constant Folding

The optimizer evaluates constant expressions at compile time so your generated code runs faster.

🔧

Functions

Define typed functions with parameters and return types. Type-checked at the call site.

🖨️

Print & Trace

Simple print for output. trace shows variable values alongside expressions for debugging.

🎯

Compiles to JavaScript

Piece+ generates clean, readable JavaScript — so your programs run anywhere Node.js does.

Static analysis

Static Constraints

Piece+ enforces these rules at compile time — before your program ever runs.

⚠️

Type Checking

Variables must match their declared type. Assigning a string to an int is a compile error.

⚠️

Undeclared Variables

Using a variable before declaring it is caught immediately with a clear error message.

⚠️

Duplicate Declarations

Declaring the same variable twice in the same scope is not allowed.

⚠️

Function Argument Types

Arguments passed to functions must match the declared parameter types.

⚠️

Argument Count

Calling a function with too few or too many arguments is a compile-time error.

⚠️

Return Type Matching

A function's return value must match its declared return type.

⚠️

Missing Return

Every function with a return type must have a return statement.

⚠️

Expression Type Compatibility

Binary expressions require compatible types. You cannot add an int to a string.

⚠️

String Comparisons

Strings cannot be compared with <, >, <=, or >=.

⚠️

Undeclared Functions

Calling a function that hasn't been defined is caught at compile time.
